Fusiondb nightly builds

These installers are built each night from the develop branch of the eXist GitHub: https://github.com/exist-db/exist. New builds will only appear when the source base actually changes, that means there may not be a build for each day if there were no changes committed on that day. The JAR files are signed by me and are provided to the community for convenience. All builds require Java 8 and include all eXist extensions.
